# Abkuerzungen

| Abkuerzung | Bedeutung | Kurzkontext |
|---|---|---|
| IC | In Character | Handlung in der Welt |
| OOC | Out of Character | Meta-Hinweis ausserhalb der Welt |
| GM | Game Master | Spielleitung |
| Co-GM | Co-Game-Master | Unterstuetzende Spielleitung |
| LE | Lebensenergie | Belastung und Verletzbarkeit |
| AE | Astralenergie | Ressource fuer Magie/Ritual |
| RS | Ruestungsschutz | Schadensminderung |
| XP | Erfahrungspunkte | Fortschritt |
| PvE | Player vs Environment | Konflikt gegen Weltlage/NSC |
| PvP | Player vs Player | Konflikt zwischen Figuren |
| WIP | Work in Progress | Inhalt in Bearbeitung |
| PR | Pull Request | Aenderungsvorschlag im Repo |

Kurzregel:

- Lieber einmal ausschreiben als dreimal raetseln.
- Neue Kuerzel nur einfuehren, wenn sie wirklich wiederkehren.
