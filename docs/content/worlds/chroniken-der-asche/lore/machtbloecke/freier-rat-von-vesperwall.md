# Freier Rat von Vesperwall

Vesperwall ist keine saubere Hauptstadt.  
Es ist eine Linie aus Türmen, Dreck und zornigen Kompromissen.

Der Freie Rat entstand, weil niemand mehr auf Hilfe aus den Zentren warten wollte.  
Kommandanten, Lagermeister, Stadtsprecher: alle brachten etwas, keiner bekam alles.

## Wie der Rat funktioniert

Beschlüsse gelten, solange sie tragen.  
Wer Nachschub bringt, bekommt Salzstimme.  
Wer nur redet, verliert sie beim nächsten Frost.

Jede neue Stimme schwört über dem Salzteller-Eid.  
Ein Tropfen Blut, ein Korn Salz, ein Name.

Die Drei-Schlag-Laterne entscheidet nachts über Zugangsrecht.  
Drei Schläge heißen offen. Zwei heißen warten. Einer heißt laufen.

Ich kann im Rat gehört werden, wenn ich etwas Reales sichere:  
einen Pass, einen Brunnen, eine Nacht ohne Überfall.

## Woran er scheitert

Der Rat ist stark gegen aussen.  
Innen frisst ihn Misstrauen und die Rostbrotquote.

Die Rostbrotquote klingt klein.  
Sie entscheidet trotzdem, wer im vierten Winter noch spricht.

Nerez bietet Kredite.  
Die Glutrichter bieten Ordnung.  
Beides klingt verlockend, wenn der Winter früh kommt.
